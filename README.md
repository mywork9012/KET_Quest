# 🎯 KET 冲刺背词软件

> 四年级 · 已过剑桥三级 · 4个月闭环冲刺 KET  
> 词 + 句绑定记忆，艾宾浩斯智能复习，120天备考计划

---

## 功能列表

| 模块 | 说明 |
|------|------|
| 📅 今日计划 | 120天备考计划，自动分配每日新词+复习 |
| 📖 学新词 | 单词卡片 + KET标准例句 + 掌握标记 |
| 🔄 复习单词 | 艾宾浩斯遗忘曲线，智能安排复习 |
| ✏️ 填空练习 | 看中文写英文单词，KET完形填空风格 |
| 🧩 选词练习 | 四选一选词，同词性干扰项 |
| 📊 我的进度 | 掌握统计 + 薄弱词清单 + 话题进度 |
| 📋 词库浏览 | 按话题/级别/状态筛选，支持搜索 |

---

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## Streamlit Cloud 部署

1. 将本项目推送到 GitHub（公开仓库）
2. 访问 https://share.streamlit.io
3. 填写仓库地址，主文件选 `app.py`
4. 点击 **Deploy** 即可

> **注意**：`data/ket_progress.db` 是 SQLite 进度数据库，  
> Streamlit Cloud 每次重启会重置。如需持久化，可升级到  
> `st.secrets` + Supabase/PostgreSQL（后续按需扩展）。

## 词库说明

- **总词数**：198词（含KET官方词汇表核心高频词）
- **A类**（40词）：剑桥三级重合词，快速巩固
- **B类**（118词）：KET核心必考词，每词2句KET例句
- **C类**（40词）：KET拓展加分词，每词1句例句

例句规范：
- 句长 6–12 词，时态以现在/过去/将来时为主
- 词汇严格控制在KET考纲内
- 场景覆盖：校园、家庭、购物、出行、运动、天气

## 数据文件

```
ket_app/
├── app.py                  # 主应用
├── requirements.txt
├── .streamlit/config.toml  # 主题配置
├── utils/
│   ├── db.py               # SQLite数据层 + 艾宾浩斯算法
│   └── planner.py          # 120天备考计划
└── data/
    ├── words.json           # 198词词库（含例句）
    ├── build_words.py       # 词库生成脚本（已执行）
    └── ket_progress.db      # 用户进度（自动生成）
```
